# NLW Operator

NLW Operator é um projeto desenvolvido durante o evento **Next Level Week #22** da [Rocketseat](https://www.rocketseat.com.br), focado em **Desenvolvimento Agêntico e Inteligência Artificial**.

O evento foi realizado em março de 2026 e apresentou três trilhas diferentes, cada uma resultando em um projeto completo:

## Projetos

### 1. DevRoast (Full-Stack)
Aplicação web de análise de código que fornece feedback brutalmente honesto (ou em modo "roast" sarcástico).

- **Tech Stack**: Next.js 16, Tailwind CSS v4, tRPC, Drizzle, Biome
- **Features**: Submissão de código, modo roast, análise detalhada com sugestões de correção, leaderboard
- **Repo Original**: [rocketseat-education/nlw-operator-fullstack-devroast](https://github.com/rocketseat-education/nlw-operator-fullstack-devroast)

### 2. ClipMaker (Iniciante)
Aplicação web que usa IA para analisar vídeos e identificar automaticamente os momentos com maior potencial de viralização.

- **Tech Stack**: HTML, Tailwind CSS, GSAP, Cloudinary, Google Gemini
- **Features**: Upload de vídeo, transcrição automática, análise de IA para identificar segmento viral

### 3. Vision Guard (Computer Vision)
Sistema de reconhecimento de gestos em tempo real usando visão computacional.

- **Tech Stack**: Python, MediaPipe, FastHTML, OpenCV, PyTorch
- **Features**: Reconhecimento de gestos via webcam, interface web com WebSocket, treinamento de modelos customizados

## Sobre o NLW

O **Next Level Week** é o maior evento de programação prática da Rocketseat. Nesta edição (#22), o foco foi em **Desenvolvimento Agêntico** - a criação de sistemas inteligentes capazes de agir de forma autônoma utilizando IA.
